# College Placement Prediction using Machine Learning

This project predicts student placement chances using Machine Learning algorithms based on various academic and skill-related factors such as CGPA, IQ, communication skills, internship experience, projects completed, and academic performance.

## Models Implemented

* Logistic Regression
* Support Vector Machine (SVM)
* Decision Tree Classification (ID3)

## Model Accuracy Comparison

| Model               | Accuracy |
| ------------------- | -------- |
| Logistic Regression | 88.7%    |
| SVM                 | 94.9%    |
| Decision Tree       | 96.3%    |

After comparing all models, the Decision Tree model achieved the highest accuracy after tuning the model to reduce overfitting.
